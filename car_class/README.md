# Car Class Program

## Description
This program demonstrates Object-Oriented Programming (OOP) and encapsulation in Python.

The program contains a `Car` class with the following private attributes:
- year_model
- make
- speed

## Features
- Accelerate the car by 5 mph
- Brake the car by 5 mph
- Display the current speed
- Uses encapsulation through private data attributes

## Files
- `car.py` - Contains the Car class
- `test_car.py` - Tests the Car class
